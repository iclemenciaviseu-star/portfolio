# Portfolio de Automacoes - Viseu Advogados

Colecao de ferramentas de automacao juridica desenvolvidas pela Controladoria Digital.

## Projetos

| Repositorio | Descricao |
|---|---|
| [rpa-judicial-viseu](https://github.com/iclemenciaviseu-star/rpa-judicial-viseu) | Automacao judicial multiportal (eProc + PJe) via Chrome CDP |
| [Publicacoes](https://github.com/iclemenciaviseu-star/Publicacoes) | Suite de leitura de publicacoes com extensoes Chrome + OCR |
| [DistribuidorAtividades](https://github.com/iclemenciaviseu-star/DistribuidorAtividades) | Sistema de gestao e distribuicao de tarefas para equipes juridicas |
| [VerificacaoCadastro](https://github.com/iclemenciaviseu-star/VerificacaoCadastro) | Auditoria automatizada de cadastros em sistemas juridicos |
| [rpa-projects-viseu](https://github.com/iclemenciaviseu-star/rpa-projects-viseu) | RPAs de cadastro (Anima, PROCON Nubank, Pandora) |
| [ComplementoCadastro](https://github.com/iclemenciaviseu-star/ComplementoCadastro) | Complementacao de dados em cadastros existentes |
